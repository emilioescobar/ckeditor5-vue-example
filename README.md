# ckeditor5 vue3 vite example

Example CKEditor 5 rich text editor for vue3, WYSIWYG editor

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
